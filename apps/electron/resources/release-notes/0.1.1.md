# v0.1.1 - Upstream v0.10.3 merge and panel stability

## Features

- **Merged upstream v0.10.3** - Brings in upstream v0.10.2 and v0.10.3 changes, including link label values, clearer Anthropic account visibility, Claude Fable 5 model support, and the Claude Agent SDK 0.3.170 update.

## Improvements

- **Workspace data isolation** - Workspace runtime data now lives under `.zo`, with workspace change handling and image resolution updated for the new storage location.
- **Default workspace sessions** - New sessions now default to the active workspace directory.

## Bug Fixes

- **Resizable panels stabilized** - Panels can resize beyond their nominal minimum widths when the layout needs to fit, while the final panel fills remaining space without storing inflated widths. This fixes the two-panel fill issue and the third-panel shrink issue seen in multi-panel layouts.
- **Workspace event handling** - Fixed `.zo` workspace change events so local workspace updates propagate reliably.

## Breaking Changes

- None.
