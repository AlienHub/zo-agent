# v0.1.5 - Working-directory files panel

## Features

- **Resizable working-directory files panel in chat** - The chat view gains a collapsible side panel that lists the files in the session's working directory, so you can browse and open project files without leaving the conversation. It's resizable, remembers its width and which folders you've expanded, and adapts to the available space - inline when the panel is wide, a floating popover when the chat gets narrow - staying correct inside multi-panel layouts.

## Improvements

- None.

## Bug Fixes

- **Opening a large JSON file no longer freezes the app** - The JSON file preview and its code-viewer fallback are not virtualized, so opening a JSON file with tens of thousands of lines hung the main thread. Size guards now collapse the JSON tree and skip the full re-parse above 256 KB, fall back to the read-only code viewer above 2 MB, and drop syntax highlighting plus the per-line gutter above 256 KB. The last guard also fixes the same freeze for large code, text, and log files.

## Breaking Changes
