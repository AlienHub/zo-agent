# Craft Agents OSS 0.1.2

- Fixed packaged Electron builds missing Pi runtime resources. The release builder now copies `pi-agent-server`, `session-mcp-server`, the target `koffi` native module, and a matching Bun runtime into the final app package during `afterPack`, so Pi subprocess sessions can start from clean CI-built apps.
- Aligned packaged Bun runtime resolution with the new app resource layout while keeping compatibility with older Windows packages.
