# v0.1.4 - Unsigned macOS builds & self-update

## Features

- None.

## Improvements

- **Self-update for unsigned macOS builds** - Unsigned macOS builds can now update themselves through a custom bundle-swap flow, and CI ships unsigned macOS artifacts with an auto-generated changelog.

## Bug Fixes

- **macOS launch on unsigned builds** - Disabled hardenedRuntime for unsigned builds so the app launches without code-signing.

## Breaking Changes

- None.
