# Zo 0.1.0

Initial Zo desktop release.

- Establishes the Zo app identity, bundle id, installer names, and GitHub release update channel.
- Publishes macOS, Windows, and Linux desktop builds from GitHub tag releases.
- Bundles the Claude Agent SDK runtime module, target-architecture native Claude binary, ripgrep, Pi subprocess server, session MCP server, target `koffi` native module, and matching Bun runtime into packaged Electron apps.
- Fails CI packaging if required runtime files are missing.
